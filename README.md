# List of the movies

## Developer

### Install

Install `pre-commit` hook:
```shell
pre-commit install
```

Install packages:
```shell
uv install
```

### Setup

Go to workdir:
```shell
cd .\list-of-the-movies\
```

Run dev server:
```shell
fastapi dev
```
